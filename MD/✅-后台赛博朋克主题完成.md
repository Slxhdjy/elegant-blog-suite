## 🤖 赛博朋克主题特色

### 视觉效果
- **霓虹配色** - 蓝色 (#00f0ff) 和粉色 (#ff006e) 的霓虹灯配色
- **光晕效果** - 文字、边框、按钮的发光效果
- **深色背景** - 深蓝黑色科技感背景 (#0a0e27)
- **高对比度** - 白色文字配合霓虹色，确保可读性
- **渐变效果** - 霓虹色渐变按钮和卡片

### 交互效果
- **悬停光束** - 卡片悬停时的光束扫过动画
- **霓虹按钮** - 按钮悬停时颜色变化和光晕增强
- **表格高亮** - 行悬停时的霓虹背景效果
- **滚动条** - 霓虹渐变滚动条

## 🔧 自定义建议

如果你想调整赛博朋克主题的效果，可以修改以下变量：

```css
/* 在 blog/css/themes.css 中 */
:root[data-theme="cyberpunk"] {
    --primary-color: #00f0ff;    /* 改变主色调 */
    --secondary-color: #ff006e;  /* 改变次要色 */
    --bg-color: #0a0e27;         /* 改变背景色 */
}
```

## 📝 注意事项

1. **性能优化** - 光晕效果使用了多层阴影，在低端设备上可能影响性能
2. **可读性** - 深色背景确保了文字的可读性
3. **一致性** - 前后台使用相同的主题变量，保持视觉一致
4. **兼容性** - 所有现代浏览器都支持这些CSS特性
