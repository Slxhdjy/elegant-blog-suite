## 🎯 切换到粉色主题

### 前台切换

#### 方法A：浏览器控制台
```javascript
// 打开任意前台页面，按F12打开控制台，输入：
window.themeManager.switchTheme('pink');
```

#### 方法B：直接设置
```javascript
document.documentElement.setAttribute('data-theme', 'pink');
localStorage.setItem('blog_theme', 'pink');
// 刷新页面生效
```

### 后台切换

#### 方法A：后台管理系统
1. 登录后台管理系统
2. 进入"系统设置"页面
3. 找到主题选择器
4. 选择"🌸 粉色浪漫"
5. 保存设置

#### 方法B：浏览器控制台
```javascript
// 在后台页面按F12打开控制台，输入：
window.adminThemeManager.switchTheme('pink');
```

## 🔧 自定义调整

### 修改主色调
编辑 `blog/css/themes.css`：

```css
:root[data-theme="pink"] {
    /* 修改这里的颜色值 */
    --primary-color: #f48fb1;  /* 改成你喜欢的粉色 */
    --primary-dark: #d16a8a;   /* 对应的深色 */
    --primary-light: #fce4ec;  /* 对应的浅色 */
}
```

### 调整导航栏透明度
```css
:root[data-theme="pink"] {
    /* 修改透明度（0.7 = 70%透明） */
    --nav-bg: rgba(209, 106, 138, 0.7);
}
```

### 修改阴影强度
```css
:root[data-theme="pink"] {
    /* 调整阴影大小和透明度 */
    --shadow-md: 0 4px 12px rgba(244,143,177,0.12);
}
```

## 🐛 常见问题

### Q1: 主题切换后没有生效？
**A**: 清除浏览器缓存并刷新页面（Ctrl+F5）

### Q2: 导航栏颜色不对？
**A**: 确保引入了 `themes.css` 和 `modern-nav.css`

### Q3: 前后台主题不同步？
**A**: 前后台使用独立的主题设置，需要分别切换

### Q4: 移动端显示异常？
**A**: 检查是否引入了 `modern-nav.js` 文件

### Q5: 想恢复默认主题？
**A**: 切换到"🌊 蓝色海洋"主题即可

## 💡 使用技巧

### 技巧1：快速切换主题
在浏览器书签栏添加以下代码：
```javascript
javascript:(function(){window.themeManager.switchTheme('pink');})();
```

### 技巧2：主题预设
为不同页面设置不同主题：
```javascript
// 在页面加载时根据路径设置主题
if (location.pathname.includes('gallery')) {
    window.themeManager.switchTheme('pink');
}
```

### 技巧3：时间自动切换
```javascript
// 白天使用粉色主题，晚上使用深色主题
const hour = new Date().getHours();
if (hour >= 6 && hour < 18) {
    window.themeManager.switchTheme('pink');
} else {
    window.themeManager.switchTheme('dark');
}
```
